It is the year 2017, most adults in the United States uses a mobile device to access the internet. No matter what kind of device or browser you are using, everyone wants to use a responsive site. A responsive website can make businesses run smoothly and make the internet more entertaining and interactive. If a website is not responsive it will lead its users to move onto another sight. When you are developing a web page it is important to check various browsers. A user should be able to tell what items you could click on, or touch if they are using a touchscreen.
